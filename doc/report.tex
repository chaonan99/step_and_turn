\section{Step-and-turn游戏规则}
该游戏是作者自创的。规则为：
\begin{itemize}
	\item 两名游戏者轮流行动；
	\item 棋盘上有棋子和障碍物，游戏者每人控制一些棋子；
	\item 每个棋子有四个方向，棋子可能的行动为沿当前方向向前一步（step）和通过左转、右转变换方向（turn）；
	\item 轮到某位游戏者时，游戏者先选择一枚棋子进行移动，后选择一枚棋子进行旋转，所选棋子可以相同，也可以不同；
	\item 移动后不得与障碍物、本方其他棋子和对方棋子重叠；
	\item 率先无法移动的游戏者判为失败。
\end{itemize}
